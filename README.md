# 💰 BudgetLife — Data-Driven Analytics Dashboard

**Smart Expense Optimization Platform — 12-Algorithm Consumer Analytics & Prediction Engine**

## 📋 Overview
BudgetLife is an AI-powered financial management platform. This dashboard analyzes 2,000 Indian survey respondents using 12 ML algorithms across 4 analytical layers.

## 🧠 Algorithms (12)
| # | Algorithm | Category |
|---|-----------|----------|
| 1 | Decision Tree | Classification |
| 2 | Random Forest | Classification |
| 3 | Logistic Regression | Classification |
| 4 | Gradient Boosting | Classification |
| 5 | Naive Bayes | Classification |
| 6 | KNN | Classification |
| 7 | K-Means | Clustering |
| 8 | Hierarchical (Ward) | Clustering |
| 9 | Apriori | Association Rules |
| 10 | Ridge Regression | Regression |
| 11 | ARIMA | Forecasting |
| 12 | Exponential Smoothing | Forecasting |

## 📊 Dashboard Tabs (9)
1. 📊 Overview & Descriptive
2. 🔍 EDA & Data Preparation Pipeline
3. 🔬 Clustering (K-Means + Hierarchical + Dendrogram)
4. 🔗 Association Rule Mining (Apriori)
